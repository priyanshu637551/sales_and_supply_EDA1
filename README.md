# Superstore Sales & Supply Chain EDA

## Project Overview
This project performs an Exploratory Data Analysis (EDA) on a Superstore dataset to understand sales trends, profit margins, and supply chain efficiency. 

Uniquely, this project includes a **custom data generation script** to simulate realistic retail scenarios, followed by data cleaning in **Python** and visualization in **Power BI**.

## File Structure
* **`superstore_fake_data.py`**: A Python script using the `Faker` library to generate the custom dataset used for this analysis.
* **`EDA_superstore.ipynb`**: Jupyter Notebook containing the data cleaning, processing, and initial Python visualizations.
* **`superstore performance.pbix`**: The interactive Power BI Dashboard file.
* **`eda_analysis_summary.csv`**: The processed data used for the final reporting.

## Tech Stack
* **Python**: Pandas, NumPy, Faker (for data generation).
* **Power BI**: For creating the interactive dashboard and KPIs.
* **Jupyter Notebook**: For documentation and code execution.

## Key Insights & Analysis
* **Data Generation**: Created a synthetic dataset to simulate complex supply chain scenarios.
* **Sales Trends**: Analyzed peak sales periods and category performance.
* **Supply Chain**: Evaluated delivery times and shipping modes.
